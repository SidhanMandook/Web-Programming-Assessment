# Digital Library Catalog

This is a simple digital library website built using HTML, CSS, and JavaScript.
It displays books in a catalog format and allows users to search, filter, and save their favorite books.

---

## Features

* Book catalog displayed in a grid layout
* Search books by title
* Filter books by genre
* Click on a book to view details
* Add and remove books from favorites
* Separate favorites view
* Dark mode toggle
* Uses localStorage to save favorites

---

## Project Structure

```
digital-library/
│── index.html
│── style.css
│── script.js
│── images/
```

---

## How to Run

Just open the `index.html` file in your browser.

---

## Notes

* All book images are stored locally in the `images` folder
* No backend is used, everything runs in the browser
* Favorites are saved using localStorage
